Возможности и преимущества

Портал разработан с использованием Angular, RxJS, NgRx и является одностраничным приложением. Для хранения и обработки необходимых данных обеспечено взаимодействие с сервером. Прототип - 21vek.by

  • Проработанный дизайн

    Единое стилистическое оформление, плавные анимации, качественные свайперы, все есть и ничего лишнего

  • Адаптивная верстка

    Поддерживаются разрешения для десктопа (1024px), планшетов (768px) и мобильных устройств (не менее 500px)

  • Интуитивный и дружелюбный интерфейс

    Проработанные стили интерактивных элементов, тултип блока "В наличии", подсказки при ошибках сети и т.д.

  • Дополнительный кастомный функционал

    Сортировка по количеству товаров, тоггл кнопок "Корзина" и "Избранные", работа с этими категориями для незарегистрированного пользователя и т.д.

  • Полное редактирование заказов

    Работа с очередью заказов, возможность их редактирования и удаления, изменения входящих позиций, их количества, иных данных заказов

  • Логичная стратегия роутинга

    Последовательная и логичная навигация, история путей, восстановление страницы роута при перезагрузке
